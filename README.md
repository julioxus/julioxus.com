# julioxus.com

Personal portfolio and cybersecurity services website.

Built with Astro 6, Tailwind CSS v4, and deployed on GitHub Pages.

## Stack

- **Framework**: Astro 6
- **Styling**: Tailwind CSS v4
- **i18n**: Custom bilingual system (EN/ES)
- **Hosting**: GitHub Pages
- **DNS**: Cloudflare
- **Contact**: Formspree

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Deployment

Pushes to `main` trigger automatic deployment via GitHub Actions.
